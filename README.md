- method-override

https://www.npmjs.com/package/method-override

- ejs
일반 JavaScript로 HTML 마크업을 생성할 수 있도록 해주는 라이브러리
https://ejs.co/

- passport
Node.js용 인증 미들웨어 모듈
https://www.passportjs.org/
